---
title: 场景视觉设计
description: POST /api/agent/scenes
---

从策划/剧本文本中识别所有物理地点，按地点维度去重，为每个唯一场景生成空镜图提示词。

支持多轮修改。

---

## 请求

**`POST /api/agent/scenes`**

### 请求体

<ParamField body="planning" type="string" required>
  剧本正文或策划文本，包含场景、地点、情节等信息。
</ParamField>

<ParamField body="film_style" type="string" default="写实风格">
  画风提示词，追加在每个场景生图词末尾。
</ParamField>

<ParamField body="lang" type="string" default="zh">
  输出语言：`zh`（中文）或 `en`（英文）
</ParamField>

<ParamField body="feedback" type="string">
  修改意见，多轮修改时传入。
</ParamField>

<ParamField body="previous_response_id" type="string">
  多轮修改时传入上一次 `done` 事件中的 `response_id`。首次生成不传此字段。
</ParamField>

### 示例请求

<CodeGroup>

```bash 首次生成
curl -X POST http://localhost:8000/api/agent/scenes \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": "第一集：宗门广场，弟子集会。王府书房，王爷审问女主...",
    "film_style": "写实风格"
  }'
```

```bash 多轮修改
curl -X POST http://localhost:8000/api/agent/scenes \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": "...",
    "film_style": "写实风格",
    "feedback": "宗门广场加入夜晚灵气氛围",
    "previous_response_id": "resp_0217bbbb"
  }'
```

```python Python
import httpx

with httpx.stream("POST", "http://localhost:8000/api/agent/scenes", json={
    "planning": "...",
    "film_style": "写实风格",
}) as r:
    for line in r.iter_lines():
        if line.startswith("data: "):
            print(line[6:])
```

</CodeGroup>

## 响应

**`200 OK`** — SSE 流式响应（`text/event-stream`）

### chunk 事件

```
data: {"chunk": "[{\"sceneId\": 1..."}

data: {"chunk": ", \"sceneName\": \"宗门广场\"..."}
```

### 响应字段结构

<ResponseField name="scene_designs" type="object[]">
  场景视觉设计列表（已按地点去重）

  <Expandable title="场景字段">
    <ResponseField name="sceneId" type="integer">场景序号</ResponseField>
    <ResponseField name="sceneName" type="string">场景名称（物理地点）</ResponseField>
    <ResponseField name="sceneDescription" type="string">简要环境说明</ResponseField>
    <ResponseField name="scenePrompt" type="string">
      场景空镜图生图词。固定以 `广角镜头，俯视角度，没有任何人物，` 开头
    </ResponseField>
  </Expandable>
</ResponseField>

---

## 注意事项

- 场景以**物理地点**为单位，同一地点只生成一张，不按镜头片段拆分
- 所有场景图均为**无人空镜**，不包含人物
- `scenePrompt` 固定前缀不可修改：`广角镜头，俯视角度，没有任何人物，`
