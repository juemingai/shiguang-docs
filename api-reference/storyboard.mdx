---
title: 分镜脚本
description: POST /api/agent/storyboard
---

将剧本策划拆解为逐帧分镜脚本，包含景别、画面描述、对白/旁白、时长建议，以及每帧的 AI 图片/视频生成提示词（中英文双语）。

---

## 请求

**`POST /api/agent/storyboard`**

### 请求体

<ParamField body="planning" type="object" required>
  完整策划内容，来自 `/api/agent/planning` `done` 事件中的 `planning` 字段。至少需包含 `premise`、`outline`、`episodes` 三个字段。
</ParamField>

<ParamField body="characters" type="object[]">
  角色/场景设计结果，来自 `/api/agent/character` `done` 事件的完整响应（包含 `characters` 和 `scenes`）。传入可让分镜提示词与角色/场景设计保持一致。
</ParamField>

<ParamField body="episode_number" type="integer">
  指定生成哪一集的分镜脚本。不传则生成全部集数。
</ParamField>

### 示例请求

<CodeGroup>

```bash 生成指定集分镜
curl -X POST http://localhost:8000/api/agent/storyboard \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": {
      "premise": "落魄千金穿越古代，邂逅冷峻王爷，在阴谋与爱情中浴火重生",
      "outline": "女主穿越后身份成谜被王爷收留，从对立到相爱，联手破敌...",
      "episodes": [
        {
          "episode_number": 1,
          "title": "穿越惊魂",
          "summary": "林若曦意外穿越，被王爷救下却被当成奸细",
          "hook": "王爷发现女主身份破绽，冷声：你到底是何人？",
          "key_scene": "女主坠入枯井，被王爷一把拉住的瞬间"
        }
      ]
    },
    "episode_number": 1
  }'
```
