---
title: 角色视觉设计
description: POST /api/agent/characters
---

从策划大纲的人物小传中提取角色信息，生成标准化的角色生图词，包含从头到脚的完整外貌描述及画质词。

支持多轮修改。

---

## 请求

**`POST /api/agent/characters`**

### 请求体

<ParamField body="planning" type="string" required>
  策划文本，包含人物小传。通常为 `/api/agent/planning` 生成的完整策划内容（可传 Markdown 或 JSON 字符串）。
</ParamField>

<ParamField body="film_style" type="string" default="写实风格">
  画风提示词，追加在每个角色生图词末尾。

  示例：`"古风插画风格"`、`"3D渲染风格"`、`"写实摄影风格"`
</ParamField>

<ParamField body="lang" type="string" default="zh">
  输出语言：`zh`（中文）或 `en`（英文）
</ParamField>

<ParamField body="feedback" type="string">
  修改意见，多轮修改时传入。
</ParamField>

<ParamField body="previous_response_id" type="string">
  多轮修改时传入上一次 `done` 事件中的 `response_id`。首次生成不传此字段。
</ParamField>

### 示例请求

<CodeGroup>

```bash 首次生成
curl -X POST http://localhost:8000/api/agent/characters \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": "人物小传：林若曦，女主，22岁，现代律师意外穿越，外柔内刚，清丽脱俗，黑发如瀑，着月白广袖长裙...",
    "film_style": "写实风格"
  }'
```

```bash 多轮修改
curl -X POST http://localhost:8000/api/agent/characters \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": "...",
    "film_style": "写实风格",
    "feedback": "林若曦的服装改为红色嫁衣，增加头饰描述",
    "previous_response_id": "resp_0217aaaa"
  }'
```

```python Python
import httpx

with httpx.stream("POST", "http://localhost:8000/api/agent/characters", json={
    "planning": "人物小传：...",
    "film_style": "写实风格",
}) as r:
    for line in r.iter_lines():
        if line.startswith("data: "):
            print(line[6:])
```

</CodeGroup>

## 响应

**`200 OK`** — SSE 流式响应（`text/event-stream`）

### chunk 事件

```
data: {"chunk": "{\"character_designs\": [{\"id\": 1..."}

data: {"chunk": ", \"name\": \"林若曦\"..."}
```

### done 事件

```json
data: {
  "done": true,
  "response_id": "resp_0217aaaa",
  "character_designs": [
    {
      "id": 1,
      "name": "林若曦",
      "gender": "女",
      "generate_image": true,
      "character_generate_prompt": "人类，黄种人，女，青年，20-25岁外观，高挑匀称，瓜子脸，无胡须，黑色长发盘髻，月白广袖汉服，绣花布鞋，单角色，完整全身照，双手自然垂放，正视图。没有其他元素的纯白背景。中性表情。杰作，最高质量，超精细细节，清晰对焦，8K分辨率，写实风格"
    }
  ]
}
```

### 响应字段结构

<ResponseField name="character_designs" type="object[]">
  角色视觉设计列表

  <Expandable title="角色字段">
    <ResponseField name="id" type="integer">角色序号</ResponseField>
    <ResponseField name="name" type="string">角色名称</ResponseField>
    <ResponseField name="gender" type="string">`男` / `女`</ResponseField>
    <ResponseField name="generate_image" type="boolean">
      是否需要生成图片。旁白、系统音等非实体角色为 `false`，其余为 `true`
    </ResponseField>
    <ResponseField name="character_generate_prompt" type="string">
      完整角色生图词。结构为：外貌描述 + 画面质量词 + 画风提示词
    </ResponseField>
  </Expandable>
</ResponseField>

---

## 生图词结构说明

生成的 `character_generate_prompt` 严格按从头到脚顺序描述，末尾追加固定画质词：

| 维度 | 示例 |
|------|------|
| 物种与种族 | 人类，黄种人 |
| 性别 | 女 |
| 视觉年龄 | 20-25岁外观 |
| 体型身材 | 高挑匀称 |
| 面部特征 | 瓜子脸，无胡须 |
| 眼部 | 丹凤眼，黑色瞳孔 |
| 发色发型 | 黑色长发盘髻，簪花发饰 |
| 服装 | 月白广袖汉服 |
| 鞋履 | 绣花布鞋 |
| 固定画质词 | 单角色，完整全身照…杰作，最高质量，8K分辨率 |
| 画风提示词 | 写实风格 |
