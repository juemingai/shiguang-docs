---
title: 剧本正文（短片）
description: POST /api/agent/film-screenplay
---

根据影视策划，生成短片电影的完整剧本正文，以 Markdown 格式流式输出，包含场景描述、人物动作、对白。

支持多轮修改。

---

## 请求

**`POST /api/agent/film-screenplay`**

### 请求体

<ParamField body="planning" type="string" required>
  影视策划的 Markdown 文本，来自 `/api/agent/film-planning` 生成结果的格式化版本。
</ParamField>

<ParamField body="feedback" type="string">
  修改意见，多轮修改时传入。
</ParamField>

<ParamField body="previous_response_id" type="string">
  多轮修改时传入上一次 `done` 事件中的 `response_id`。首次生成不传此字段。
</ParamField>

### 示例请求

<CodeGroup>

```bash 首次生成
curl -X POST http://localhost:8000/api/agent/film-screenplay \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": "# 最后一班列车\n\n**片长**：5分钟\n\n**主题**：亲情与遗憾\n\n## 三幕结构\n\n..."
  }'
```

```bash 多轮修改
curl -X POST http://localhost:8000/api/agent/film-screenplay \
  -H "Content-Type: application/json" \
  -N \
  -d '{
    "planning": "...",
    "feedback": "第二幕节奏太慢，压缩回忆戏份，加强父亲的内心独白",
    "previous_response_id": "resp_0217zzzz"
  }'
```

```python Python
import httpx

with httpx.stream("POST", "http://localhost:8000/api/agent/film-screenplay", json={
    "planning": "# 最后一班列车\n\n...",
}) as r:
    for line in r.iter_lines():
        if line.startswith("data: "):
            print(line[6:])
```

</CodeGroup>

## 响应

**`200 OK`** — SSE 流式响应（`text/event-stream`）

剧本正文以 Markdown 格式流式推送，每个 `chunk` 为剧本文本片段。

### chunk 事件

```
data: {"chunk": "# 最后一班列车\n\n"}

data: {"chunk": "## 第一幕\n\n**内景 绿皮火车车厢 — 清晨**\n\n"}

data: {"chunk": "昏黄的灯光打在斑驳的座椅上。父亲（58岁）独坐窗边，"}

data: {"chunk": "手里攥着一封泛黄的信封，目光空洞地望向窗外飞逝的田野。\n\n"}

data: {"chunk": "> 父亲（自语）：十年了……\n\n"}
```

### done 事件

全部内容生成完毕后推送，仅包含 `response_id`，无额外结构化数据。

```json
data: {"done": true, "response_id": "resp_0217zzzz"}
```

### 响应字段结构

<ResponseField name="done" type="boolean">
  固定为 `true`，标识生成完成
</ResponseField>

<ResponseField name="response_id" type="string">
  本次生成的 ID，可用于下一次多轮修改的 `previous_response_id`
</ResponseField>

## 剧本格式说明

生成的剧本正文遵循标准电影剧本格式，以 Markdown 呈现：

| 元素 | 格式示例 |
|------|---------|
| 场景标题 | `**内景 地点 — 时间**` |
| 动作描述 | 普通段落文字 |
| 对白 | `> 角色名：台词内容` |
| 幕标题 | `## 第一幕` |
